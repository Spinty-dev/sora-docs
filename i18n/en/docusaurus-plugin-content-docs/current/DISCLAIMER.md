## 6. Special Technical Means (STM) / Clandestine Surveillance
SORA is not intended to be used as a "Special Technical Means" (STM) for clandestine information gathering or secret interception. 
- The software is designed for transparent, interactive security auditing. 
- It does not contain features for "spyware", does not mimic everyday household objects, and is not intended for hidden (covert) installation. 
- SORA works exclusively with standard network interfaces (Wi-Fi adapters) and does not contain or require hardware modifications that turn the original device into an STM.
- Traffic obfuscation features (StealthEngine) are provided exclusively for testing the resilience of intrusion detection systems (WIDS/IDS) within a legitimate audit framework and are not designed for clandestine interception of private data.
