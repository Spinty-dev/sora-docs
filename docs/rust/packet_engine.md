# Packet Engine

`PacketEngine` отвечает за обработку входящего трафика с `wlan0mon`.

## Основные этапы
1. Чтение сырого буфера `AF_PACKET` (нулевые аллокации, если пакет не валиден).
2. Парсинг 802.11 заголовков.
3. Валидация кадра (FCS, PMKID, EAPOL-маркеры).
4. Отправка отфильтрованных ценных EAPOL-фреймов в Python через MPSC.
