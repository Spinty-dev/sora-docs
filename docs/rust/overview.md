# Ядро Rust (High-Performance Layer)

Этот раздел документации описывает архитектуру высоконагруженной сетевой части (Сниффер и Инжектор), написанной на Rust.

Главные задачи Rust-ядра:
- Обработка `AF_PACKET` raw-сокетов.
- On-the-fly сборка `4-way handshake` без дропов.
- Удержание state-машины WPA3 (SAE) в памяти.
- Очереди отправки пакетов с приоритетами.
