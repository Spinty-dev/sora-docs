- Укрепления безопасности беспроводных сетей.

## 6. Специальные технические средства (СТС)
SORA (Signals Offensive Radio Auditor) не является специальным техническим средством (СТС) для негласного получения информации в смысле **ст. 138.1 УК РФ**. 
- Программное обеспечение предназначено для открытого аудита безопасности беспроводных сетей.
- Оно не содержит скрытых функций для «шпионажа», не имитирует бытовые предметы и не предназначено для скрытого (нерегламентированного) внедрения. 
- SORA работает исключительно со стандартными сетевыми интерфейсами (Wi-Fi адаптерами) и не содержит и не требует аппаратных модификаций, превращающих исходное устройство в СТС.
- Функции маскировки трафика (StealthEngine) служат исключительно для тестирования устойчивости систем обнаружения вторжений (WIDS/IDS) в рамках легитимного аудита и не предназначены для негласного перехвата конфиденциальной информации.
