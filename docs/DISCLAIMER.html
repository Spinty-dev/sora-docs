Программное обеспечение предоставляется «КАК ЕСТЬ» (AS IS), без каких-либо гарантий, явных или подразумеваемых. Автор не гарантирует бесперебойную работу ПО и не несет ответственности за любой ущерб, прямой или косвенный (включая порчу оборудования, потерю данных или финансовые потери), возникший в результате использования или невозможности использования SORA.

4. Этичное использование​

Проект создан исключительно в целях:

  • Образования и изучения протоколов беспроводной связи.
  • Проведения легитимных тестов на проникновение (Penetration Testing).
  • Укрепления безопасности беспроводных сетей.

5. Экспортный контроль и товары двойного назначения​

SORA (Signals Offensive Radio Auditor) содержит функционал, который может быть классифицирован как «товары и технологии двойного назначения» в соответствии с международными правилами (Вассенаарские соглашения) и законодательством РФ (включая экспортный контроль ФСТЭК России).

  • Использование данного ПО может быть ограничено правилами экспорта, реэкспорта или импорта в вашей юрисдикции.
  • Вы несете единоличную ответственность за соблюдение всех применимых законов об экспортном контроле.
  • Данное ПО НЕ предназначено для использования в юрисдикциях, где подобные инструменты классифицируются как контролируемое оружие или спецсредства без надлежащего государственного разрешения.

6. Специальные технические средства (СТС)​

SORA (Signals Offensive Radio Auditor) не является специальным техническим средством (СТС) для негласного получения информации в смысле ст. 138.1 УК РФ.

  • Программное обеспечение предназначено для открытого аудита безопасности беспроводных сетей.
  • Оно не содержит скрытых функций для «шпионажа», не имитирует бытовые предметы и не предназначено для скрытого (нерегламентированного) внедрения.
  • SORA работает исключительно со стандартными сетевыми интерфейсами (Wi-Fi адаптерами) и не содержит и не требует аппаратных модификаций, превращающих исходное устройство в СТС.
  • Функции маскировки трафика (StealthEngine) служат исключительно для тестирования устойчивости систем обнаружения вторжений (WIDS/IDS) в рамках легитимного аудита и не предназначены для негласного перехвата конфиденциальной информации.

1. Authorized Use Only​

Using SORA (Signals Offensive Radio Auditor) to gain unauthorized access to computer networks or devices, intercept traffic, or conduct DoS attacks on networks for which you do not have express written permission to perform a security audit is illegal.

2. User Responsibility​

The user bears sole and full responsibility for any actions performed using this software. It is the user's responsibility to comply with all applicable local, state, and international laws, including but not limited to:

  • United States: Computer Fraud and Abuse Act (CFAA), 18 U.S.C. § 1030.
  • United Kingdom: Computer Misuse Act 1990.
  • Europe: Directive 2013/40/EU on attacks against information systems.
  • International: Relevant local cybercrime and telecommunications regulations.

3. No Warranty​

The software is provided "AS IS", without warranty of any kind, express or implied. The author shall not be liable for any damages (including hardware damage, data loss, or financial loss) arising from the use or inability to use this software.

4. Ethical Use​

This project is intended for:

  • Educational purposes and wireless protocol research.
  • Legitimate Penetration Testing.
  • Hardening wireless network security.

5. Dual-Use and Export Control (International)​

SORA (Signals Offensive Radio Auditor) contains capabilities that may be classified as "Dual-Use" items under international regulations, including the Wassenaar Arrangement.

  • Accessing or using this software may be subject to export, re-export, or import regulations in your jurisdiction.
  • You are responsible for ensuring that your acquisition and use of SORA comply with all applicable export control laws.
  • This software is NOT intended for use in jurisdictions where such tools are prohibited or classified as controlled weapons/munitions without proper government authorization.

6. Special Technical Means (STM) / Clandestine Surveillance​

SORA is not intended to be used as a "Special Technical Means" (STM) for clandestine information gathering or secret interception.

  • The software is designed for transparent, interactive security auditing.
  • It does not contain features for "spyware", does not mimic everyday household objects, and is not intended for hidden (covert) installation.
  • SORA works exclusively with standard network interfaces (Wi-Fi adapters) and does not contain or require hardware modifications that turn the original device into an STM.
  • Traffic obfuscation features (StealthEngine) are provided exclusively for testing the resilience of intrusion detection systems (WIDS/IDS) within a legitimate audit framework and are not designed for clandestine interception of private data.
